La structure de base d'un document HTML
===

```html
<!DOCTYPE html><!-- doctype HTML5 -->
<!-- element racine -->
<html>
  <head>
    <!-- titre -->
    <title>Document sans titre</title>
    <!-- encodage des caractères -->
    <meta charset="utf-8">
  </head>
  <body>
    <!-- corps du document -->
  </body>
</html>
```

> Indentez votre code !
